# O Jogo do Dono — Project Context

## Visão Geral

Jornada empresarial gamificada de 7 dias para empresários. Preço: R$297. Objetivo: fazer o empresário executar missões reais, diagnosticar gargalos da empresa e, ao final, converter para o Gestão de Impacto.

## Frase-Mãe

> "7 dias para provar que tua empresa aguenta crescer."

> "Você não joga para ganhar o GI. Você joga para provar que merece sentar na mesa."

## Público-Alvo

Donos de empresa e empresários. Muitos com pouca familiaridade com plataformas digitais. Acesso majoritariamente mobile. Precisam de UX simples, intuitivo, com poucos cliques.

## Direção Visual

- Dark mode premium
- Paleta: preto, grafite, dourado, off-white, cinza claro, vermelho escuro apenas para alertas, verde apenas para sucesso
- Visual tático, empresarial, misterioso — painel de missão, não área de membros
- Sem estética de joguinho adolescente

## Tom de Voz

Direto, provocativo, empresarial, adulto, com senso de consequência. Sem motivacional barato.

Usar frases como:
- "O que não é medido cobra juros."
- "A empresa não escala onde o dono centraliza."
- "Clareza sem ação vira culpa organizada."
- "Dono bom não tenta pular processo."

## Missões da Jornada

| Dia | Nome | Tema |
|-----|------|------|
| 0 | O Raio-X | Diagnóstico inicial |
| 1 | O Cofre | Financeiro / vazamentos |
| 2 | O Espelho | O dono como gargalo |
| 3 | O Campo de Batalha | Equipe |
| 4 | A Máquina Quebrada | Processos |
| 5 | A Caçada | Vendas — ação em 24h |
| 6 | A Sentença | Plano de ação 30 dias |
| 7 | O Conselho Final | Decisão de compra do GI |

## Estados das Missões

- **Bloqueada** — ainda não liberada
- **Liberada** — disponível para execução
- **Em andamento** — usuário iniciou
- **Concluída** — missão entregue
- **Atrasada** — prazo passou, ainda pode entregar com penalidade
- **Enviada para análise** — aguardando validação do admin

## Estrutura Padrão de Página de Missão

1. Header: nome da missão, dia, status, pontuação possível, contagem regressiva
2. Frase provocativa
3. Vídeo do dia
4. Objetivo + instruções
5. Formulário específico do dia
6. Checklist de execução
7. Upload de evidência
8. Botão "Concluir Missão"
9. Mensagem pós-conclusão

## Sistema de Pontuação

- Missão no prazo: 100 pts
- Profundidade da resposta: até 150 pts
- Evidência enviada: 200 pts
- Resultado financeiro/comercial: até 300 pts
- Participação em live: 100 pts
- Concluir tudo: 500 pts bônus

## Conversão Final

- Passaporte GI: desbloqueado ao concluir missões
- Botão 1: compra direta (checkout)
- Botão 2: falar com Conselheiro GI (WhatsApp)
- A Mesa dos Donos: almoço presencial como bônus de status

## MVP Prioritário

Login → Dashboard → Mapa da jornada → 8 páginas de missão → bloqueio diário → formulários → envio de evidência → pontuação → ranking → Passaporte GI → admin básico
